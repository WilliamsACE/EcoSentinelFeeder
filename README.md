## 🧠 About
EcoSentinel Feeder is an intelligent system designed to provide **smart nutrition** for animals in need, optimizing feeding through efficient and sustainable technology.

<br />

## 🌟 Features
- 🤖 **AI-Powered Detection:** Identifies dogs and cats using computer vision before dispensing food.
- 📡 **Real-Time Dashboard:** Built with Django to monitor feeding data and ESP32 camera feeds.
- 🌱 **Sustainable & Autonomous:** Designed to run efficiently on a Raspberry Pi environment.
